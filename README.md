# アーユルヴェーダ体質診断アプリ

アーユルヴェーダの三大ドーシャ（ヴァータ・ピッタ・カパ）に基づく体質診断アプリケーションです。

## 機能

- **体質診断**: 睡眠時間、食事の好み、気分の傾向から主体質を判定
- **季節別ヨガポーズ提案**: 現在の季節とドーシャに応じたポーズを提案
- **時間帯別ヨガポーズ提案**: 現在の時間帯に最適なポーズを提案
- **食べ物アドバイス**: おすすめ食材と控えめにする食材
- **起床・就寝時間の提案**: ドーシャに合った生活リズム
- **1日の過ごし方**: 朝・昼・夜のアドバイス

## 使い方

1. アプリにアクセスすると、季節と時間帯が自動検出されます
2. 睡眠時間、食事の好み、気分の傾向を入力します
3. 「診断する」ボタンをクリックすると結果が表示されます
4. 「もう一度診断する」ボタンで再診断できます

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- 外部ライブラリ不使用
- 単一HTMLファイルで動作
